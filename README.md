# 📚 智慧职教 自动学习助手

自动完成在线课程（智慧职教）中的视频、音频、PPT 课件学习任务，支持自动跳转和遍历，解放双手。

---

## ✨ 功能特性

- **智能检测**：自动识别当前页面中的视频、音频、PPT 等学习资源。
- **自动跳过**：
  - 视频：强制跳转到末尾（支持 iframe 嵌套）。
  - 音频：跳转到末尾并锁定进度，防止被网页恢复。
  - PPT：自动翻页直到最后一页（支持“下一页”按钮检测）。
- **下一个课件**：自动点击“下一个”链接，进入下一节课程。
- **自动运行模式**：循环执行“跳过当前课件 → 进入下一个课件”，直到所有课程完成。
- **手动/自动选择标签页**：可手动选择浏览器标签页，也可自动识别智慧职教的课程页面。
- **多平台适配**：主要针对智慧职教（zjy2.icve.com.cn），也兼容Bilibili 等视频网站。
Tips：强烈不推荐用于超星学习通，因为跳过也拿不到任务点

---

## 🖥️ 环境要求

- **操作系统**：Windows（依赖 `win32gui`/`win32process`）
- **浏览器**：Microsoft Edge（需已安装）
- **Python 版本**：3.6 及以上

## 📦 安装与依赖

### 1. 克隆或下载项目

1. 安装 Python 依赖
建议使用虚拟环境（可选）：

bash
pip install requests websocket-client pywin32
注意：pywin32 提供了 win32gui 和 win32process 模块，必须安装。

完整依赖列表（requirements.txt）：
requests>=2.25.0
websocket-client>=1.2.0
pywin32>=305

一键安装：
bash
pip install -r requirements.txt

🚀 使用方法
1. 启动程序
bash
python zjy.py
程序会自动启动（或连接到）Edge 浏览器，并开启远程调试端口（9222）。

2. 操作流程
登录平台：在打开的 Edge 浏览器中，登录智慧职教（或超星等），进入课程学习页面。

选择模式：
自动选择模式：程序自动查找智慧职教的课程页面，无需手动选择。
手动选择模式：列出所有标签页，输入编号选择目标页面。
执行命令（在控制台中输入）：
命令	说明
video -skip	强制跳转当前视频到末尾
audio -skip	强制跳转当前音频到末尾
ppt -skip	自动翻页到 PPT 最后一页
-skip	智能跳过：优先视频 → 音频 → PPT
next	点击“下一个课件”链接
autorun	自动循环：-skip → next，直到无下一个课件
-r	返回主菜单，重新选择模式
quit	退出程序
🛠️ 打包为 EXE（可选）
使用 PyInstaller 打包成独立的可执行文件，方便在没有 Python 环境的电脑上运行：

bash
pip install pyinstaller
pyinstaller -F zjy.py
打包后的 zjy.exe 会出现在 dist 文件夹中。

⚠️ 注意事项
仅限 Windows：由于使用了 win32gui，该程序只能在 Windows 上运行。
安全软件提示：程序会启动 Edge 并开启远程调试端口，部分杀毒软件可能会报警，请添加信任或暂时关闭。
不要用于超星学习通：代码作者明确提示“极其不推荐将该程序用于超星学习通”，可能因为反作弊机制，请谨慎使用。
合规使用：本工具仅用于辅助学习，请勿用于作弊或违反平台规定的行为。使用后果由用户自行承担。
浏览器窗口：程序启动的 Edge 会使用独立的用户数据目录（C:\Temp\EdgeDebug），不会影响你的日常浏览器配置。

🐛 常见问题
Q: 启动时提示“未找到 Edge 浏览器”？
A: 请确保 Edge 安装在默认路径（C:\Program Files (x86)\Microsoft\Edge\Application\msedge.exe）或修改代码中的路径。

Q: 连接调试端口失败？
A: 检查是否有其他程序占用了 9222 端口，或手动关闭所有 Edge 进程后重试。

Q: 点击“下一个课件”无效？
A: 不同平台的下一个按钮选择器可能不同，若无效可尝试手动模式或联系作者适配。
